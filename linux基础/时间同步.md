# 时间同步

**/var/spool/cron是计划任务目录，某个用户创建计划任务，就会在该目录下生成相应目录**

	yum install -y ntp
	/usr/sbin/ntpdate pool.ntp.org

	ntpdate -u NTP-SERVER

	echo '*/5 * * * * /usr/sbin/ntpdate pool.ntp.org > /dev/null 2>&1'>>/var/spool/cron/root
